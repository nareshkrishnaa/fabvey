S.No HTML Element Description
1 <html> Defines the root of an HTML document.
2 <head> Contains metadata about the document.
3 <title> Defines the title of the HTML document.
4 <meta> Provides metadata about the document.
5 <link> Specifies external resources, like stylesheets.
6 <style> Embeds CSS styles within the document.
7 <script> Embeds JavaScript code or references an external script.
8 <noscript> Provides content to be displayed if JavaScript is disabled.
9 <base> Specifies a base URL for relative URLs.
10 <body> Defines the main content of the document.
11 <h1> to <h6> Defines headings of varying levels.
12 <p> Represents a paragraph of text.
13 <a> Creates hyperlinks to other web pages or resources.
14 <img> Embeds images in the document.

16 <audio> Embeds audio content.
17 <source> Specifies multiple media resources for <video> and <audio>.
18 <iframe> Embeds an inline frame.
19 <div> Defines a division or section for styling and layout.
20 <span> Inline container for a section of text or other elements.
21 <ul> Defines an unordered (bulleted) list.
22 <ol> Defines an ordered (numbered) list.
23 <li> Represents a list item within <ul> or <ol>.
24 <dl> Defines a description list.
25 <dt> Defines a term in a description list.
26 <dd> Defines a description in a description list.
27 <table> Creates a table.
28 <caption> Defines a table caption.
29 <thead> Groups header content in a table.
30 <tbody> Groups body content in a table.
31 <tfoot> Groups footer content in a table.
32 <tr> Defines a table row.
33 <th> Defines a table header cell.
34 <td> Defines a table data cell.
35 <colgroup> Groups columns in a table for styling.
36 <col> Specifies column properties within a <colgroup>.
37 <form> Creates an HTML form for user input.
38 <input> Represents an input field within a form.
39 <button> Defines a clickable button within a form.
40 <textarea> Creates a multi-line text input field within a form.
41 <select> Creates a dropdown list within a form.
42 <label> Represents a label for an input element.
43 <fieldset> Groups form elements and labels.
44 <legend> Defines a caption for a <fieldset>.
